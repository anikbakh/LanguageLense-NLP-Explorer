# LanguageLense: NLP Explorer

An interactive Streamlit application demonstrating how AI uses Natural Language Processing (NLP) to understand human language through tokenization.

## Features

- Explore tokenization using your own English or Persian text
- Compare English and Persian tokenization side-by-side
- Learn the fundamentals of NLP through an interactive interface

## Technologies

- Python
- Streamlit
- NLTK
- Natural Language Processing (NLP)
